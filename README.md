## 服务器端渲染(ssr) Demo

### 操作步骤
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8088
npm run serve

# bulid for production with minification
npm run build
```

### 启动成功后
 > 浏览器直接访问: localhost:8088

### 安装后结构:
```
.
|--node_modules                         //项目依赖包
|--package.json                         //项目基本信息
|--README.md                            //项目说明
|--server.js                            //配置文件
|--.gitignore                           //git上传时需要忽略掉的文件
```
